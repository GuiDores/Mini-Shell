# 🐚 Mini-Shell

Um Mini-Shell desenvolvido em linguagem C para fins de estudo sobre sistemas operacionais, processos e execução de comandos.

## 🚀 Como Executar o Projeto

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:
* Compilador GCC (ou MinGW no Windows)
* Ferramenta `make`

### 🔧 Compilação

Para compilar o projeto automaticamente usando o Makefile contido no repositório, abra o terminal na pasta do projeto e execute:

```bash
make
```

Isso gerará o executável do Mini-Shell.

### 💻 Execução

Após a compilação, execute o programa com o comando:

* **No Linux/macOS:**
  ```bash
  ./mini_shell
  ```
* **No Windows (Git Bash/CMD):**
  ```bash
  ./mini_shell.exe
  ```

### 🧹 Limpeza

Para limpar os arquivos de objeto e o executável gerados após o uso:

```bash
make clean
```
